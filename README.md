# Fashion Crisis: Merge Sort Edition 🎀✨

Welcome to **Fashion Crisis: Merge Sort Edition**! Initially conceptualized as *Candies for All (DAA Project)*, this project has blossomed into an interactive, wildly glamorous, Barbie-themed educational web application. 

The primary goal of this application is to teach computer science students (or anyone really!) the mathematical execution of the **Merge Sort array algorithm** in a fun, tangible, and visually stunning interactive format!

## ✨ Features
- **Divide and Conquer Gameplay**: Experience the textbook algorithm! Players must actively click to mathematically divide the parent sequences until every array length is exactly `1`.
- **Merge Sorting Physics**: Once divided, users are prompted to properly merge the items. You must manually dictate the sorting logic by clicking on the appropriately sized element! Click wrong, and the game will correct you with visual error feedback.
- **Cinematic Cutscenes**: Integrated `.mp4` video cutscenes dynamically transition to full-viewport size as you execute algorithmic phases. 
- **Voice acting**: Built-in OS-level speech synthesis organically scans your computer to find bright, female text-to-speech narrators that cheer you on. 
- **Flawless UI/UX**: Enjoy heavily animated panels, pop-in components, frosted-glass CSS physics, and an infinitely looping glitter text aesthetic using custom keyframe sequences over Google's `Nunito` font-family.
- **Quality of Life Utilities**: Integrated Mute toggles for the background piano instrumentals and instantly accessible "Restart Level" widgets.

## ✨ Technologies Used
The entire software stack runs seamlessly in the browser:
- **Vanilla HTML5**: Semantic screen tracking.
- **Vanilla CSS3**: No libraries used! Heavy use of `@keyframes` for smooth entry animations, flexbox layouts, scaling transitions, dropshadow filters, and background gradients.
- **Vanilla JavaScript**: Pure JS handles the binary tree logic representing the array segments (simulated visually via DOM element injection and mathematical coordinate shifting). It dynamically calculates the FLIP CSS bounds and evaluates the sorting correctness.
- **HTML5 Video/Audio**: Used for seamlessly looping `animate` frames and tracking `onended` cutscene event traps.

## ✨ How to Play
1. **Clone or Download the Repository**.
2. Make sure all files (including the `assets/` directory) are in the same parent folder.
3. Simply double-click **`index.html`** to open it natively in any modern web browser (Google Chrome, Firefox, Safari, Edge).
4. **No local servers or dependencies are required!**

## ✨ Gameplay Tutorial
1. Launch the `index.html` and hit **Start Game** to view the introductory Barbie runway cutscene.
2. Read the instructions carefully. 
3. **DIVIDE PHASE**: The game will randomly populate an array of items across the runway. Click the glowing pink boxes; they divide perfectly in half according to Merge Sort recursion logic. Continue clicking until they're all in isolated boxes.
4. **MERGE PHASE**: Follow the green pulsating boxes! Look at the two arrays ready to merge. **Click the smaller candy** at the front of each array. It will correctly drop into the sorted parent box above it.
5. Win the game by perfectly merging the entire tree layout!
