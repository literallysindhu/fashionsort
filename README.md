# Candies for All - DAA Project
A standalone Pygame visualizer that turns Barbie's Candy Land Fashion Show into an interactive "Divide & Conquer" computer science story!

## Story Premise
Barbie is hosting a grand fashion show in Candy Land, attended by:
- 3 Lollipop People
- 5 Milk Chocolate People
- 6 Toffee People
- 2 Sour Candy People

She needs to hand them matching candies, but they are all shuffled! Using her skills as a CSE Engineer, she uses **Divide and Conquer (Merge Sort)** to quickly sort them on her organizer tray.

## Requirements
Please ensure you have Python and `pygame` installed.
If you don't have pygame, you can quickly install it via:
```sh
pip install pygame
```

## How to play
To launch the game, navigate into this repository logic and execute:
```sh
python main.py
```

### Controls
* **SPACEBAR** - Used to navigate through the menus and dialog screens.
* **S Key** - Starts the "Divide and Conquer" (Merge Sort) sequence!
